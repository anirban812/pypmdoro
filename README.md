# PyPmdoro

Simple Python script with notification for Pomodoro.

This script needs `notify-send` to be installed on your system.

Tested on Fedora 26 with Python 3.6.2

### How to use.

Run the script with 

```sh
$ python3 pomodoro.py
```


Enter work and break durations and start working!
